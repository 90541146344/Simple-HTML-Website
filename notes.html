HTML TAGS

Headings (h)
It is used to define Headings in HTML.
ImPortant Text (Strong)
Defines important text which is usually displayed in bold.
Quote (q)
This tag indicates that the enclosed text is a short Quotation.
Citation (Cite)
It identifies the source of quotation or creative work.
Subscript (sub)
It appears half a character below the normal line, and is sometimes rendered in a smaller font.
